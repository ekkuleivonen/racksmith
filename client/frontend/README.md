# Racksmith UI

React SPA for Racksmith. Uses Vite, TanStack Query, and shadcn/ui.

## Development

```bash
npm install
npm run dev
```

Set `VITE_API_TARGET` if the API runs elsewhere (default: http://localhost:8000).

## Build

```bash
npm run build
```

Output in `dist/`. Serve with any static file server; proxy `/api` and WebSockets to the backend.
