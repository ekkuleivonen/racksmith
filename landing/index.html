Racksmith

Ansible automation for your homelab, managed through Git. Runs on your local network — your data never leaves your LAN.

Run it

terminal 
docker run -d --network host \
  -e APP_URL=http://localhost:8080 \
  -v ./data:/app/data \
  -v ./workspace:/app/workspace \
  ghcr.io/ekkuleivonen/racksmith-client:0.1.1

Open http://localhost:8080
